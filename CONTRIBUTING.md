# Contributing

Thank you for contributing.  

## Ways to contribute

You can report a bug.  
You can request a feature.  
You can improve documentation.  
You can submit a pull request.  

## Development principles

Keep the driver dashboard simple.  
Prefer reliable data capture over extra user interface features.  
Treat location and artifact data as sensitive.  

## Bug reports

Include your browser name and version.  
Include your operating system.  
Include steps to reproduce.  
Include expected behavior and actual behavior.  

## Pull requests

Keep changes small.  
Write a clear description.  
Update documentation when behavior changes.  
Avoid adding new external dependencies unless necessary.
